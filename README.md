1. 克隆本项目到本地
    ```
    git clone https://github.com/khuqen/django-test.git
    ```
2. 新建虚拟环境
    ```
    cd django-test
    python3 -m venv env
    source env/bin/activate
    pip install django
    ```
3. 运行开发环境
    ```
    python manage.py runserver
    ```
4. 输入网址
    ```
    http://127.0.0.1:8000/polls
    ```
